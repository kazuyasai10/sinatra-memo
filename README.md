## 何ができるものなのか
ローカルのjsonファイルにメモ（id,title,content)を保存、読み込み削除ができるアプリです。

## 構築やインストール方法

```
% git clone https://github.com/kazuyasai10/sinatara-memo.git
% bundle install --path vendor/bundle
% bundle exec ruby app.rb
=> http://localhost:4567
```
